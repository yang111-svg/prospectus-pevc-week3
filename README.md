# Week 3: IPO招股说明书股本变化抽取项目

## 项目概述
本项目围绕8家IPO公司的招股说明书，建立人工gold standard和自动化抽取流程。

## 目录结构
- data/ - PDF文件清单
- manual_gold/ - 人工gold standard
- pipeline/ - 自动化流程脚本
- prompts/ - LLM提示词
- outputs/ - 自动输出
- evaluation/ - 评估结果

## 8家样本公司
603418 友升股份、301563 云汉芯城、301581 黄山谷捷、688758 赛分科技、688775 影石创新、920100 三协电机、920116 星图测控、001282 三联锻造
